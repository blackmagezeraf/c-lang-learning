Header file for Body Mass Index (BMI) calculations.

Function to Calculate Body Mass Index (BMI).

Parameters
ageThe age of the person in years.
weightThe weight of the person in kilograms.
heightThe height of the person in centimeters.
Note
The height is divided by 100.0f to convert it from centimeters to meters. This is necessary because BMI is calculated using height in meters.
The BMI formula is weight divided by the square of height in meters. This is the standard formula used to calculate BMI.
Returns
Returns the Calculated BMI
